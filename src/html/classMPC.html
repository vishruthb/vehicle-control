Detailed Description

Base MPC class. Whereas the model that you pass in takes care of specifying the dynamics and the cost function, the MPC object sets up the bounds on variables (as specified in Bounds object), and on constraints (enforces x(0) = x0, x(t+1) - f(x(t),u(t)) = 0), and also warmstarts your solver with a suitably shifted version of the last run's solution.
